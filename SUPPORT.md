# Support

If you need help using ArchGuard AI, please check the following resources:

## 1. Documentation
Please carefully read our [README.md](README.md) which contains detailed explanations on how to configure both the **Free Zero-Trust Gateway** and **Bring Your Own Key (BYOK)** modes.

## 2. GitHub Issues
If you encounter a bug, an architectural review that seems wildly incorrect, or a tool crash, please open a [GitHub Issue](https://github.com/archguard-labs/action/issues). 

When filing a bug, please include:
- Your `.github/workflows/archguard.yml` configuration.
- The specific error output from the GitHub Actions console.
- **Do not** include any API Keys or Secrets in your issue report!

## 3. Discussions
For feature requests, architectural debates, or to share how ArchGuard AI caught a sneaky bug in your repo, join us in [GitHub Discussions](https://github.com/archguard-labs/action/discussions).
